# Geek App

Sobre
--
Construa sua própria equipe de vingadores, baseando-se nos heróis (e vilões) do universo da marvel!

Como usar
--
Para instalar as depedências back end (Laravel, Guzzle...) :
```
composer install
```

Para o front end:
```
npm install
```

O que foi usado
--
Laravel, Guzzle, Vuejs, SASS e a Marvel API
